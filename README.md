# 🌟 Tarot Card of the Day

A slot machine style game to draw a random tarot card of the day.

Spin three reels. Match all three on the payline to reveal your card and its reading. A wild symbol on any reel counts as a match.

**Live demo:** [gnuboss.github.io/tarot-card-of-the-day](https://gnuboss.github.io/tarot-card-of-the-day)

---

## Features

- All 22 major arcana cards
- Wild symbol on each reel to assist matching
- Card reading revealed on match
- Zero dependencies — single HTML file, loads instantly
- Mobile friendly

## Usage

Open `index.html` in any browser. No installation or internet connection required after download.

## Hosting on GitHub Pages

1. Fork or clone this repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)` folder
4. Your live URL will be `yourusername.github.io/tarot-card-of-the-day`
